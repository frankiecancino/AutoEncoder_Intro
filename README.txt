Follow these steps

1) Install the virtualenv package on you machine, e.g., "pip install virtualenv".
2) Create a virtual enviorment, e.g., "virtualenv -p python3 lecture_24_env"
3) Activate source, e.g., "source lecture_24_env/bin/activate"
4) Install the package requirements provided requirements.txt, e.g., "pip install -r requirements.txt"
5) Run jyputer notebook, e.g., "jyputer notebook"
6) Launch the AutoEncoder Inclass notebook
