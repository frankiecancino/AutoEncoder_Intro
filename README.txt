Follow these steps

1) Install the virtualenv package on you machine, e.g., "pip install virtualenv".
2) Create a virtual enviorment, e.g., "virtualenv -p python3 venv"
3) Activate source, e.g., "source venv/bin/activate"
4) Install the package requirements provided requirements.txt, e.g., "pip install -r requirements.txt"
5) Run jupyter notebook, e.g., "jupyter notebook"
6) Launch the AutoEncoder notebook
